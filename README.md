# mager_adb
adb asissting script.
Script ini dibuat karena kadang saya mager copy paste path panjang dan ngetik ulang pas pakai adb dan deploy frida ke perangkat.

## Kompatibilitas
* Python 3
* Windows
* requirment tqdm,requests,lzma

## Fitur
1. Basic apk view.
2. Pull & Push.
3. Frida deploy & Run.
